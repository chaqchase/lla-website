# LLA Google Search Plugin

Search Google with live autosuggestions, history management, and clipboard support.

## Features

- Live Google search suggestions
- Search history management
- Clipboard fallback support
- Search from history
- Copy queries to clipboard
- Search statistics
- Top searches tracking
- Configurable history size

## Configuration

Configuration file location: `~/.config/lla/google_search/config.toml`

```toml
remember_search_history = true
use_clipboard_fallback = true
max_history_size = 100

[colors]
success = "bright_green"
info = "bright_cyan"
warning = "bright_yellow"
error = "bright_red"
prompt = "bright_blue"
```

## Usage

### Perform a Google search

```bash
lla action google_search search
```

This will:
1. Prompt you to choose input method:
   - Enter new search query (with live Google suggestions)
   - Search from history
   - Search clipboard content
2. Fetch live suggestions as you type
3. Open the search in your default browser
4. Save the query to history

### Manage search history

```bash
lla action google_search history
```

Options:
- Search selected entry
- Copy queries to clipboard
- Delete selected entries
- Clear all history
- Show search statistics

### Configure preferences

```bash
lla action google_search preferences
```

Adjust settings:
- Toggle search history
- Toggle clipboard fallback
- Set max history size

### Display help

```bash
lla action google_search help
```
